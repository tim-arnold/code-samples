# Code Samples

A collection of code samples from recent projects demonstrating WordPress development and GitHub Actions CI/CD workflows.

## Contents

### [WordPress Theme Pages](wp-theme-pages/)

Custom team member single and archive templates with Advanced Custom Fields. The single view displays a hero image, credentials, bio, and social links. The archive renders a card grid grouped by ACF category with defined section ordering.

### [WordPress Custom Blocks](wp-custom-blocks/)

Two custom Gutenberg blocks built with ACF and registered via `block.json`: a tabbed logo grid with keyboard navigation and ARIA support, and an interlocking card grid with a staggered CSS Grid layout and AJAX pagination.

### [WordPress Custom Plugin](wp-custom-plugin/)

A plugin for managing global call-to-action sections. Provides a rich ACF-based interface for configuring up to 5 CTAs with priority ordering, color scheme selection, and per-page visibility controls.

### [GitHub Actions Workflows](github-actions/)

CI/CD, backup, migration, and disaster recovery workflows covering automated deployments, scheduled backups, cross-environment migration, and database restore procedures.
